## Getting Started

This project is a Work in Progress and uses AI to solve the cube. 

### Installation

```bash
npm install
# or
yarn install
```

### Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

#### Results

Click Shuffle to Shuffle the Cube

Click Solve to auto to solve the cube
